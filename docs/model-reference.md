## How to Use

1. Go to WP Admin > Products > Models
2. Click a model term (e.g., "110")
3. Rename it to the scooter name (e.g., "Xiaomi Mi4 Lite")
4. The slug stays as "110" — the `_ewheel_model_id` meta ensures future syncs still match

As more products are imported, more model IDs will appear. Cross-reference with
ewheel.es catalog to identify the scooter for each numeric ID.

Model names will become clearer once a full sync (all products) is run, since
specific accessories often mention the scooter name directly.
